## Overview

Enabling this action provides summarized issue information from a custom view at your chosen frequency.

During configuration, you can provide a prompt for OpenAI to generate the issue summaries.

You can find the whole code for this action [here](https://github.com/vantikhq/vantik/tree/main/actions/issue-view-summary)
