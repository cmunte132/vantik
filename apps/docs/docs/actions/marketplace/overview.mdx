This section showcases the use cases we've implemented, ready for you to deploy and use in Vantik. We're dedicated to making Vantik your all-in-one solution for issue tracking. Have an idea for a new action? We'd love to hear from you! Simply create a GitHub issue to share your suggestions—we value your feedback!

## General Actions
These are the actions which an event gets triggered from any changes inside vantik.


- **[Bug Enricher](/docs/actions/marketplace/bug-enricher)** - Comment a potential solution for issues labeled as `bug` using OpenAI


  - **[Issue View Summary](/docs/actions/marketplace/issue-view-summary)** - Summarize key issue updates in a customized view, delivered at your chosen frequency


  ## Integration Based Actions
 Actions that gather data and authenticate with third-party tools like GitHub.

 
- **[Email](/docs/actions/marketplace/email-actions)** - Create issues from email in Vantik Triage
