Actions are core of Vantik. They help you write and automate workflows. 

:::note
We use [Trigger](https://trigger.dev) to manage and deploy the actions you write.
:::

Before digging deeper into the details of writing actions, you should read the [fundamentals of actions](/docs/actions/core-concepts) to understand what actions are and how they work.
