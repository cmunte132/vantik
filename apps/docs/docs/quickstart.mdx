## Overview

Vantik is the dev-first issue tracking software powered by the open-source community. It is simple, light-weight and customisable.
### Cloud

Vantik cloud is currently in public beta, you can signup [here](https://app.vantik.dev/auth)

### Self-hosted

If you prefer to self-host Vantik on your own machine, you can do so using Docker. However note that you will need to add configurations for email, AI and storage services. 
Please refer to the documentation [here](/docs/oss/self-deployment)
