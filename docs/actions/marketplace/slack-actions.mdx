## Overview

Automate Issue Creation from slack with Emoji 👀:
    - Assign 👀 emoji to a Slack thread to automatically trigger Vantik to create a triage issue.
    - This feature empowers even non-Vantik users to easily create issues.


<Tip>You can find the whole code for this action [here](https://github.com/vantikhq/vantik/tree/main/actions/slack)</Tip>
